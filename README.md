# DELILAH RESTO API STARTER KIT

> Esto es un staterter kit base para el prpyecto de delilah API

# Requerimientos

- Node.js v14+
- MySQL V5+ o Maria DB V10+

# Instalacción

### Paso 1: Clonar proyecto

> Se debe clonar el proyecto en la maquina donde se quiere levantar

```shell
git clone https://github.com/francamara/Orders-controller.git
```

### Paso 2: instalar dependecias

```shell
npm install
```

### paso 3: Configurar .env

> **Importante:** Se debe configurar el `.env`, para esto pueden copiar el archivo `.env.example`.
> Se debe configurar en el archivo `.env` los datos de conexion a la base de datos.

# Iniciar

### Desarrollo

```shell
npm run dev
```

### Producción

```shell
npm start
```
